# Switch Store - Releases

Este repositório contém apenas os binários compilados do Switch Store.

## Download

Vá para a [página de Releases](https://github.com/HayatoG/switch-store-releases/releases) para baixar a versão mais recente para seu sistema operacional.

### Plataformas Disponíveis

- 🐧 **Linux** - `ci_transfernx-linux.zip`
- 🪟 **Windows** - `ci_transfernx-windows.zip`
- 🍎 **macOS** - `ci_transfernx-macos.zip`

## Sobre

CI TransferNX - Nintendo Switch Transfer Tool

Uma ferramenta para transferência de arquivos para o Nintendo Switch.
